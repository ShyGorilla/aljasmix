# Hash Cracker

## What is Hash Cracker
This tool can crack hashes then print it to you. 

## Types of Hashes available
1- md5
2- sha1
3- sha224
4- sha256
5- sha384
6- sha512

## Install and start The tool 

`git clone https://github.com/ShyGorilla/hash_cracker.git`
`cd hash_cracker`
`python3 hash.py`
